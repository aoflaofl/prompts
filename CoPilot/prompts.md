# CoPilot Prompts

## Refactoring prompts

```text

/explain and identify any code smells

Identify the parts of this method that could be refactored into smaller separate methods

Refactor this method and move the part for [description] to a separate method

Refactor and extract the repetitive code into a new method

Refactor this method into smaller methods implementing the Single Responsibility Principle

```

## Some other refactoring ideas

```text

update this code to use features available in Java 11

suggest other data structures that could handle this task more efficiently

optimize this code for better performance

rename the parameters and variables in this method to reflect their purpose

```
